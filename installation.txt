Reinforcement Learning for ALM
ALM-RL

Required libraries
- ffmpeg
- cuda
- cudnn
- gymnasium
- sb3
- Pytorch

Mirroring the repository: 
	git push --mirror https://github.com/Merijn68/rl-alm.git
Cloning the repository
	mkdir rl-alm
	cd rl-alm
	git clone https://github.com/Merijn68/rl-alm.git
Create the environment
	conda create -p .\rl-alm # This makes sure the environment is stored in the same directory as the source code.
	conda activate .\rl-alm # This was neccesarary as I had no more space on C:

Remove a conda environment (Is something goes wrong).
	conda remove --name rl-alm --all
